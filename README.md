# Pokedex App

A Pokémon explorer app built with **React** and **PokéAPI**.  
Search, filter, and view detailed information about Pokémon with a clean and responsive UI.

## 🚀 Features
- 🔍 Search Pokémon by name
- 📖 View Pokémon stats, types, and abilities
- 📱 Fully responsive design
- ⚡ Powered by React + API integration

## 🛠️ Tech Stack
- React.js
- PokéAPI
- CSS / Tailwind
